## プロンプトで仕事が劇的に変わる

ChatGPTに「メールを書いて」と雑に指示しても、あまり使えない文章しか出てきません。**プロンプトの質が、そのまま出力の質になります。**

この記事では、筆者が実際に仕事で毎日使っている、コピペで即使えるプロンプト20個を紹介します。すべてテストして効果を確認済みです。

## 基本の構造

使えるプロンプトには共通の型があります：

```
【役割】
あなたは○○の専門家です。

【タスク】
以下の内容について○○してください。

【入力】
{ここに情報を貼り付け}

【制約・出力形式】
- 文字数は○○字以内
- 文体は○○
- 出力は○○形式で
```

この型を覚えておくだけで、どんな業務でも応用できます。

## メール関連プロンプト

### 1. ビジネスメール作成

```
あなたは経験豊富なビジネスパーソンです。
以下の要件に沿って、ビジネスメールを作成してください。

【要件】
- 相手: 取引先の担当者
- 目的: 打ち合わせ日程の調整
- 希望日: 4/20 10:00-、4/22 14:00-
- トーン: 丁寧だが堅すぎない

【制約】
- 件名も含める
- 300字以内
- 最後に「ご確認のほどよろしくお願いいたします」で締める
```

### 2. 返信の下書き

```
以下の受信メールに対する返信の下書きを作成してください。
私は{自分の立場}です。返信では{伝えたいこと}を明確にしてください。

【受信メール】
{メール本文を貼り付け}

【返信の方向性】
- 結論から書く
- 丁寧だが簡潔に
- 次のアクションを提示する
```

### 3. お詫びメール

```
以下の状況に対するお詫びメールを作成してください。
誠意が伝わり、かつ次のアクションが明確な内容にしてください。

【状況】
{何が起きたか}

【今後の対応】
{どう対応するか}
```

## 文章作成プロンプト

### 4. ブログ記事の構成案

```
「{テーマ}」についてブログ記事の構成案を作成してください。

【条件】
- ターゲット: {誰向けか}
- 文字数目安: 3000字
- 目的: {何を伝えたいか}
- トーン: {どんな雰囲気か}

【出力形式】
見出し（H2）とその下の小見出し（H3）まで。各見出しで書く内容を1-2行で補足。
```

### 5. キャッチコピー生成

```
以下の商品・サービスのキャッチコピーを10個提案してください。

【商品・サービス】
{詳細}

【ターゲット】
{誰向けか}

【トーン】
{どんな雰囲気か（例：シンプル、親しみやすい、プロフェッショナル）}

各コピーは15字以内で、特徴が一目でわかるようにしてください。
```

### 6. 文章の校正

```
以下の文章を校正してください。

【チェック項目】
1. 誤字脱字
2. 文法の誤り
3. 冗長な表現
4. わかりにくい表現

【出力形式】
- 修正版の文章
- 主な修正点のリスト（各1行）

【原文】
{文章を貼り付け}
```

### 7. 要約

```
以下の文章を、以下の条件で要約してください。

【条件】
- 文字数: 300字以内
- 対象読者: 中学生でもわかるレベル
- 重要なポイント3つを箇条書きで最後に追加

【原文】
{文章を貼り付け}
```

## 議事録・会議関連

### 8. 議事録の整形

```
以下のメモから、整理された議事録を作成してください。

【出力形式】
# 会議タイトル
## 日時
## 参加者
## 議題
## 決定事項
## アクションアイテム（担当者・期限付き）
## 次回会議の予定

【メモ】
{ざっくりしたメモを貼り付け}
```

### 9. アジェンダ作成

```
以下の目的に沿った会議のアジェンダを作成してください。

【会議の目的】
{何のための会議か}

【参加者】
{誰が参加するか}

【所要時間】
60分

各議題に時間配分を付けてください。
```

## 企画・戦略プロンプト

### 10. 企画書の骨子

```
以下のアイデアについて、企画書の骨子を作成してください。

【アイデア】
{アイデアを説明}

【出力項目】
1. 背景・課題
2. 目的
3. 提案内容
4. 期待される効果（定量的に）
5. 実施スケジュール
6. 必要なリソース
7. リスクと対策
```

### 11. SWOT分析

```
{会社・サービス名}についてSWOT分析を実施してください。

【前提情報】
{事業内容・規模}

【出力形式】
表形式で、各項目3つずつ挙げてください。
- Strengths（強み）
- Weaknesses（弱み）
- Opportunities（機会）
- Threats（脅威）

最後に、この分析から導かれる戦略を3つ提案してください。
```

### 12. 競合分析

```
{自社サービス}の競合分析をしてください。

【自社サービス】
{概要}

【出力形式】
主な競合3社を挙げて、以下の観点で比較表を作ってください。
- 料金
- 主な機能
- ターゲット層
- 差別化ポイント

最後に、自社が取るべき戦略を提案してください。
```

## データ・分析プロンプト

### 13. データの解釈

```
以下のデータから、インサイトを3つ抽出してください。

【データ】
{数値データを貼り付け}

【出力形式】
1. インサイト（何が読み取れるか）
2. なぜそうなっているのかの推測
3. 次に取るべきアクション

各インサイトごとに上記3点をまとめてください。
```

### 14. アンケート集計

```
以下のアンケート自由記述回答を分析してください。

【回答】
{自由記述を貼り付け}

【分析内容】
1. 主なテーマを3つに分類
2. 各テーマの代表的な意見（引用）
3. 全体の傾向
4. 改善提案
```

## アイデア発想

### 15. ブレインストーミング

```
「{テーマ}」について、斬新なアイデアを10個提案してください。

【制約】
- 実現可能性は一旦無視してOK
- 一般的すぎるアイデアは避ける
- 各アイデアは1-2行で簡潔に

【評価】
最後に、10個の中から「最も面白い」と思うものを1つ選び、理由を教えてください。
```

### 16. 命名（ネーミング）

```
以下の商品・サービスに名前を付けてください。

【対象】
{商品・サービスの説明}

【条件】
- カタカナ/英語/漢字のいずれか
- 覚えやすく、意味が連想しやすい
- 4〜8文字程度

10個提案して、それぞれ由来も教えてください。
```

## 学習・自己成長

### 17. 概念の解説

```
「{トピック}」について、中学生にもわかるように説明してください。

【条件】
- 専門用語は使わない（使う場合は必ず解説）
- 身近な例えを入れる
- 500字以内
```

### 18. 比較して理解

```
「{A}」と「{B}」の違いを教えてください。

【出力形式】
1. それぞれの定義（50字ずつ）
2. 主な違い（3つの観点から表形式で）
3. どちらを選ぶべきかの判断基準
4. 初心者におすすめなのはどちらか
```

```
4. セキュリティの問題

```

```
```

### 20. エラーの原因解析

```
以下のエラーの原因と解決策を教えてください。

【エラー】
{エラーメッセージ}

【状況】
{どういう操作でエラーが出たか}

【環境】
{使用している言語・フレームワーク・バージョン}

考えられる原因を複数挙げ、それぞれの解決策を具体的なコードで示してください。
```

## まとめ

プロンプトは「AIとの会話の設計図」です。良い設計図があれば、良い結果が出ます。

**今日から使えるコツ:**
1. 上記のテンプレートをブックマークする
2. 自分の業務に合わせて少しずつカスタマイズする
3. うまくいったプロンプトは自分専用のドキュメントに保存する

プロンプトは資産です。1つ作れば一生使えます。ぜひ自分だけのプロンプト集を育ててください。
